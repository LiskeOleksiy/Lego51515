Claw Bot

 
This robot car has a claw that can pick up the large red ball from the ground, carry it, and drop it off somewhere else. The distance sensor and the color sensors can both be used to help the robot find target locations.
You can run the robot on either carpet or hard floors. On hard floors, a small stand can be used to keep the ball still before you pick it up. Two goals can be used as target locations to drop off the ball (or pick it back up again).
Two example autonomous programs are provided to show basic driving and turning operations and the use of the distance and color sensors to locate ball positions. There is also a remote control program that lets you freely drive the robot and operate the claw.
The claw can pick up the red ball from the ground, carry it, and put it back down again.
The distance sensor can be used to locate one of the goals.
Drop off the ball into a goal.
The color sensor can be used to find a tape line that marks the location of a ball in the stand.
The color sensor under the bot can trigger when it is over the target color.

Claw Bot 30
This autonomous program uses measured movement to find the ball ahead of it and then the distance sensor to find the goal behind it to drop off the ball. See the setup in the next image.
Claw Bot 30 Setup
This is how you should arrange the items for the Claw Bot 30 program. The robot will move forward exactly 30 cm to find the ball, so you should measure that distance.
The goal should be placed directly behind the robot as shown. The distance to the robot doesn't matter because the distance sensor will be used to find it.
Claw Bot Color
This autonomous program uses the color sensor to find a blue line that marks the location of the ball directly ahead of it and then the distance sensor to find the goal behind it to drop off the ball. See the setup in the next image.
Claw Bot Color Setup
This is how you should arrange the items for the Claw Bot Color program.
Make the blue line using blue painter's tape (or something else blue), with the beginning of the line about 8 cm from the center of the ball. The line and ball should be directly in front of the robot, but the distance doesn't matter because the robot will drive until it finds the line.
The goal should be placed directly behind the robot as shown. The distance to the robot doesn't matter because the distance sensor will be used to find it.
Claw Bot RC
This remote control program allows you to drive the robot with two sliders and control the claw with two buttons.
You can control the driving speed and steering with the two sliders, and you can also make the robot pivot back and forth in place using only the horizontal slider.
Press the red button to close the claw and the white button to open it.
