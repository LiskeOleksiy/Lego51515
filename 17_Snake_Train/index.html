Snake Train

 
This robot slithers around like a snake, and it can also follow a line like a train following a track. My cats are normally unimpressed by my LEGO robots, but they found this one disturbing...
On carpet or any surface, you can drive the Snake Train by remote control, or try the autonomous program that wiggles around in a wavy oval pattern. On a smooth surface you can lay down a line with tape for it to follow using the line following program. See the program image descriptions for details.
I left the design of the engine and the cars plain and simple, so have fun adding to or modifying them!
Slither around through your carpet by remote control or autonomously
Set up a curved line for it to follow like a train track

Snake Train RC
The Platform Gyro progam uses the hub's gyro sensor, so see note at the end of this page to make sure it is calibrated first.
This remote control program allows you to drive the Snake Train with a simple switch and slider. You can't back up, and it only goes full speed, so you need to keep steering!
Snake Train Oval
This autonomous program makes the Snake Train drive in a repeating curvy oval pattern.
You can also make your own autonomous programs using just movement blocks, but when you string multiple movement blocks together the robot stops after each one. This program keeps it constantly moving. Also, this program uses the gyro sensor to make the turning angles more accurate.
Snake Train Follow
This autonomous program makes the Snake Train follow a line on the ground using the color sensor (in reflected light mode).
To make a line, use dark tape if your floor is light colored. Black electrical tape is best because it is flexible and you can make curves if you gently stretch it while laying it down. If your floor is dark colored, you can use white tape. Make your curves gradual, the train will not be able to follow tight turns.
Start the program with the light sensor centered directly over the line. The program will scan the line and the floor to measure the light reflection of each and then start following the line.
 

Gyro Sensor Problem and Workaround
The built-in gyro sensor in the LEGO MINDSTORMS Robot Inventor hub allows a program to sense the angular direction and motion of the hub (in all three axes: roll, pitch, and yaw). This can be used to make a robot sense changes in its tilt or turn angle, for example.
However, for the gyro sensor to work properly, it needs to complete an internal calibration each time you power up the hub, and this calibration requires that the hub be completely motionless for a few seconds. Until the calibration has a chance to complete, the gyro sensor will just report 0 angle no matter what direction it is facing.
So, if you are having trouble getting a program that uses the gyro sensor to work, especially if it doesn't seem to respond at all, or if the model behaves erratically and totally different from the intended design, set the model down on a table or floor and let go, so that the hub is totally motionless for a few seconds. This should allow the gyro to complete its calibration and start working.
